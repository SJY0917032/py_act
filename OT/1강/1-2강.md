## JS    
    jQuery
        굳이 jQuery로만 만드는것은   
        구시대의 방법으로만 만드는것과 동일한방법.
        html
        css
        JS (매우 중요해짐)
    Heavy/JS
        브라우저 호환성을 생각하면 ES5를 사용해야한다
        -> webpack등을써서 코드변환을하면 OK
          
        UI 프레임워크/라이브러리가 필요하다
        React.js, Vue.js, Angular를 쓰면 된다.
    JS
        거의 모든 브라우저에서 H/C/JS를 지원
    NodeJS
        데스크탑/서버단 개발이 가능
        윈도우에서의 MS진영의 전폭적인 지지
    JS ver.
        ES(no)로 버전이 구분된다
        최소 ES6로 개발하는것이 효율성 높게 개발이 가능하다.
